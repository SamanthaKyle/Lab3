# GGR472 Lab 3 - Map Interactivity and Controls

Created an interactive map of Toronto showing the existing Toronto Heat Relief network, using points to represent relief sites and showing wards coloured according to number of sites.
Created using Bootstrap and Mapbox.

## Layer with data/symbology classified:
Ward layer fill colour is based on number of heat relief sites per ward

## Pop-up window
Pop-ups will appear on click of any heat-relief site to give site name, address, and type of heat-relief feature

## Method that changes with map visualization
The feature-types are invisible at the wide and default zoom level, but zooming in closer reveals them

## A method that fileters map features based on an event
The ward currently hovered-over appears bolder than other wards using a mouse-event and filter combination, as shown in class

## Map controls and HTML features
Fullscreen button in top right of map container

Legend for ward styling on left
